All I18n message or text should follow below convention, for e.g. if TT is Tool Tip so this I18n message is for Tool Tip.  
For creating any new I18n message kindly follow convention. {NAMING CONVENTION}_{DESCRIPTION}. For e.g. TT_NAME 
	 
Naming Convention: Kindly add any new naming convention and its description if you found on the way. 
If possible try to keep it of 2 Chars if possible. 
		-- TT : Tool Tip. 
		-- CA : Caption.
		-- PH : Place Holder.
		-- TXT : Text written or used in the HTML.
		-- MSG : Message for alert, warning or popup etc.
		-- TTL : Title Or Header
		-- TH  : Table Header
